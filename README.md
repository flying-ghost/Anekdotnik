Название проекта: Анекдотник

Цель проекта: создать простой консольный скрипт на Python, который случайным образом выдаёт афоризмы/анекдоты с сайта anekdot.ru. 
Позволяет расслабиться и поднять настроение в пару кликов!

Функционал:
Автоматически загружает случайный анекдот с сайта ;
Выводит анекдот в консоли с цветной подсветкой (использует библиотеку colorama);
Даёт возможность запросить новый анекдот нажатием Enter;
Позволяет завершить работу скрипта командой stop.

Используемые библиотеки:
requests — для отправки HTTP-запросов на сайт;
BeautifulSoup (bs4) — для парсинга HTML-страницы и извлечения текста анекдота;
colorama — для цветной раскраски вывода в консоли.

Установка и запуск:
Убедитесь, что у вас установлен Python.
Тут есть файлик с зависимостями поэтому командой:
pip install --upgrade -r requirements.txt

Запустите скрипт в терминале:
python main.py 

Как пользоваться:
После запуска вы увидите приветственное сообщение и первый анекдот.
Для получения нового анекдота просто нажмите Enter.
Чтобы завершить работу программы, введите stop и нажмите Enter.

Ограничения и особенности:
Зависит от доступности сайта anekdot.ru;
Если сайт недоступен или элемент с анекдотом не найден, скрипт вернёт шуточное сообщение;
Работает только в консольном режиме;
Использует парсинг сайта — при изменении структуры HTML может потребовать доработки.

Безопасность:
Скрипт не собирает и не передаёт никаких личных данных. Работает только с публичным контентом сайта.

Спасибо за использование! Надеюсь, анекдоты поднимут вам настроение!
